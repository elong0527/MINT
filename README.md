# R package

An R package for Nonparametric Microbial Interdependence Test  (NMIT)

## Installation ##

I assume that all auxillary packages are already installed

From an interactive R session:

```r
library(devtools)
install_github("elong0527/NMIT")
library(NMIT)
```
